# 太平洋西北公路旅行 · PNW Road Trip 🌲🌊

一个交互式行程网页:7 月 14–22 日,西雅图出发绕俄勒冈海岸一圈(9 天 8 晚)。
点顶部日期查看每天的路线、沿途打卡点和住宿,地图用 Google Maps。

## 使用

打开 `index.html` 即可。地图需要一个免费的 Google Maps API Key —— 在 `index.html`
顶部把 `GOOGLE_MAPS_API_KEY = ""` 的引号填上你的 key,保存刷新即可(需启用
Maps JavaScript API / Directions API / Geocoding API 三个服务)。

## 在线访问

部署在 GitHub Pages。key 已用「网站来源限制」锁定到本站域名。
